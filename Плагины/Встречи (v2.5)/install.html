INSTALLATION INSTRUCTIONS

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html, license.html, upgrade.html, and changelog.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in the "/images/" directory in BINARY mode. Upload all other files in ASCII mode.

  2. Set full permissions (CHMOD 777) on the "/uploads_event" directory and all the files in the "/templates" directory.

  3. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin you have installed. Click the update link. Your upgrade is now complete!

  4. Be sure to delete the file "admin/install_event.php" when you are finished. Leaving this file intact poses a serious security risk!
