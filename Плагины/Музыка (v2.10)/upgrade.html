UPGRADE INSTRUCTIONS

  1. Upload all of the files and folders contained in the "/files" directory in this zip to your server. Do not upload install.html, license.html, upgrade.html, or changelog.html.
    - Upload all files in the "/images/" directory in BINARY mode. Upload all other files in ASCII mode.
    - Be sure to set full permissions (CHMOD 777) on all the files in the "/templates" directory.
    Note: If you have made modifications to your SocialEngine installation, refer to changelog.html for a list of files and language variables modified in each version. If you have made changes to those files, we suggest using WinMerge to consolidate your changes with the new files. Make sure that you also upload "/admin/install_music.php".

  2. Once you have finished uploading the files to your server, log into your SocialEngine admin panel. On the "View Plugins" page, you will see the plugin you have installed. Click the upgrade link.

  3. If you have "Subdirectory URLs" enabled, go to your "URL Settings" page. Click the "Save Changes" button, then select all of the text in the box and copy it. Open a text editor such as notepad, and paste the text. Save the document as ".htaccess", and upload it to your server, overwriting the previous one.
  4. You may also want to increase the PHP upload_max_filesize and PHP post_max_size to allow for larger uploads. Some hosting providers allow you to do this via your control panel, or by editing the php.ini configuration file. Apache based servers may be able to add the following lines to the .htaccess file: php_value upload_max_filesize 12M php_value post_max_size 12M
  5. Be sure to delete the file "admin/install_music.php" when you are finished. Leaving this file intact poses a serious security risk!
  6. Your upgrade is now complete!
