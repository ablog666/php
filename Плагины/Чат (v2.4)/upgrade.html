UPGRADE INSTRUCTIONS

  1. Make a COMPLETE BACKUP before you begin.
  2. Upload all of the files and folders contained in the "/files" directory in this zip to your server. Do not upload install.html, license.html, upgrade.html, or changelog.html.
    - Upload all files in the "/images/" directory in BINARY mode. Upload all other files in ASCII mode.
    - Be sure to set full permissions (CHMOD 777) on all the files in the "/templates" directory.
    Note: If you have made modifications to your SocialEngine installation, refer to changelog.html for a list of files and language variables modified in each version. If you have made changes to those files, we suggest using WinMerge to consolidate your changes with the new files. Make sure that you also upload "/admin/install_chat.php".

  3. Login to your SocialEngine admin panel. On the "View Plugins" page you will a link that will allow you to upgrade. Click the upgrade link. Your upgrade is now complete!

  4. Be sure to delete the file "admin/install_chat.php" when you are finished. Leaving this file intact poses a serious security risk!
