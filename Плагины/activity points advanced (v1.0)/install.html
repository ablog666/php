INSTALLATION INSTRUCTIONS

  1. Upload all of the files and folders contained in this zip file (in the "/files" directory) to your server, EXCEPT for install.html, changelog.html, upgrade.html and license.html. Be sure to upload the files to the directory on your server where you have already installed SocialEngine. Upload all files in ASCII mode.


  2. Set full permissions (CHMOD 777) on the folder "/uploads_userpoints" directory.


  3. Set full permissions (CHMOD 777) on all files in the "/templates" directory.


  4. Login to your SocialEngine admin panel. On the "View Plugins" page you will see the plugin you have installed. Click the install link. Your installation is now complete!


  5. Be sure to delete the file "admin/install_userpoints.php" when you are finished. Leaving this file intact poses a serious security risk!

INTEGRATION INSTRUCTIONS

  1. Integrating User Homepage template.
    You will be inserting full contents of the file user_home_userpoints.tpl into user_home.tpl.
    Open file templates/user_home.tpl, search "SHOW LAST 3 NEWS ANNOUNCEMENTS" and copy&paste right before that line all contents of the file templates/user_home_userpoints.tpl.


  2. Integrating Site Homepage template.
    You will be inserting full contents of the file home_userpoints.tpl into home.tpl.
    Open file templates/home.tpl, search "SHOW ONLINE USERS IF MORE THAN ZERO" and copy&paste right before that line all contents of the file templates/home_userpoints.tpl.
